How WebAPI Works:
---------------
Web APIs operarte on a client-server model
The Client is an application is need the data and server is the application that has the data
the Client sends a request to the server's API endpoint,which is a specific URL where the API can be accessed

GET : read from server
POST : to create new data 
PUT: to modify existing data
DELETE : To remove data
